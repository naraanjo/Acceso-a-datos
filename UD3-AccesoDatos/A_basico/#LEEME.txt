¿QUÉ HAY EN ESTE PROYECTO?

Un ejemplo básico del uso de JPA sobre una base de datos relacional con dos tablas 1:1.
  |-> Como motor de JPA se utiliza Hibernate y como base de datos tomaremos SQLite.
  |-> El proyecto está desarrollado con Maven y Java 21.

ORDEN DE VISUALIZACIÓN
1) El fichero "persistence.xml" en la ruta "src/main/resources/META-INF/persistence.xml" define la unidad de persistencia y las propiedades de conexión a la base de datos.
